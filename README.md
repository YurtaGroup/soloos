# 🚀 Solo OS — Operating System for Solopreneurs

## Quick Start

### 1. Install Flutter SDK
```bash
# Download Flutter from https://docs.flutter.dev/get-started/install
# Then add to PATH and run:
flutter doctor
```

### 2. Create platform files
```bash
cd "Solo OS"
flutter create . --platforms=ios,android,macos,web
```

### 3. Install dependencies
```bash
flutter pub get
```

### 4. Run the app
```bash
flutter run
```

## Features

| Module | Description |
|--------|-------------|
| 🏠 Dashboard | AI Daily Digest, quick stats, habit progress |
| 📋 Work | Projects, tasks with priorities, progress tracking |
| 💪 Health | Habit tracker with streaks and mini-calendar |
| 💰 Finance | Simple income/expense tracker |
| 💡 Ideas | Max 3 active ideas pipeline with AI validation |
| 🤖 Daily Standup | AI-powered standup bot with pattern analysis |
| 🎂 Contacts | Birthday tracker with smart reminders |
| ⚙️ Settings | API key, profile, data management |

## AI Features (powered by Claude Opus 4.6)

- **Daily AI Digest** — Morning briefing with top 3 priorities
- **Standup Analysis** — Pattern recognition and coaching
- **Idea Validator** — Viability scoring and 80/20 analysis
- **Script Writer** — YouTube, Twitter, LinkedIn content
- **Executive Assistant** — Strategy and business decisions

Get your Claude API key at [console.anthropic.com](https://console.anthropic.com) and add it in Settings.
